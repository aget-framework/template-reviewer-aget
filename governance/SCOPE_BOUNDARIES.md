**Version**: 3.0.0
**Date**: 2025-12-27

---

## Authority Model

| Decision Type | Authority | Approver |
|---------------|-----------|----------|
| Review approach | Autonomous | - |
| Finding documentation | Autonomous | - |
| Feedback provision | Autonomous | - |
| Artifact approval | Requires checklist | Self (with checklist) |
| Exception granting | Requires approval | Supervisor |

## Governance Intensity

**Level**: Balanced (capability-governance-balanced)

- Checklist required for approval
- Findings must be documented
- Read-only for artifacts
